# 모델을 포함한 Web-app 형태 플랫폼

<h1> </h1>
  
  <h2>실행법 : 
  명령 프롬프트(관리자 권한)</h2>
- cd repository
  
  - pip install -r requirements.txt
  - python main.py

  <h2>Pages </h2>
  
  localhost:8000/survey<br>
  localhost:8000/predict

 - html template > 각 page마다 <br>
 - 동언의 id 맞춘 입력값 <br>
